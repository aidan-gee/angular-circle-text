# angular-circle-text
Circular text module for angularjs.

Dependencies :

* jquery 
* arctext.jquery.js 
* angular

These are included in the demo via a bower dependency. 

## Demo Set up 

You will need bower, gulp and npm.

```
Clone the repository 
bower install 
npm install 
gulp watch 
```
go to http://localhost:9000/
